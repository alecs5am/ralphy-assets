# Attribution — Italian Brainrot character images

## Source

All 33 character hero images in this directory were mirrored from **[brainrot.cyou](https://brainrot.cyou/character)** on 2026-05-11. Each image's source URL is recorded in the manifest under `pool.italian-brainrot-characters.<slug>.sourceUrl`.

## Origin of the characters

These are **AI-generated meme characters** that originated on TikTok in 2024-2025 as part of the "Italian Brainrot" viral trend. Each character was created by a TikTok user (sometimes attributed, often pseudonymous) using AI image / video generation tools (Sora, Veo, Midjourney, FLUX, etc.) paired with Italian-gibberish voiceover. Some original creator handles are noted in each character's descriptor.

The canonical wiki for these characters was hosted at `brainrot.fandom.com` until **Fandom closed it in mid-May 2025** (NSFW/moderation reasons). The image bucket was purged. `brainrot.cyou` is a third-party fan aggregator that maintains the character roster.

## Intended use in this repository

These images live here as **reference material for the [`italian-brainrot` template](https://github.com/alecs5am/ralphy/tree/main/templates/entertainment-viral/italian-brainrot)** in the ugc-cli pipeline:
- Image-to-image references for FLUX / Imagen / Gemini Image when generating new compositions featuring these characters in novel contexts
- Canonical reference for prompt-writing (so an LLM agent can match the visual style)
- NOT for direct redistribution as final video output

## License posture

These images are AI-generated meme content. Original creators are TikTok users whose authorship of AI outputs is itself an unsettled question. Treating them as **fair-use meme reference** under the framework that:

1. The use is **transformative** — references for new derivative video generation, not republication
2. The use is **attributional** — every entry in the manifest links back to its `brainrot.cyou` source page; this file credits the source
3. The use is **non-commercial in context** — open-source CLI for personal / educational video generation; if a downstream user ships commercial output featuring these characters, that responsibility transfers to them

## Take-down

If you are the original creator of one of these characters and want it removed, **please open a GitHub issue at https://github.com/alecs5am/ralphy-assets/issues** with the character slug and proof of ownership. We will remove it from the manifest and delete the image within 7 days.
